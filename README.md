## 📌 About the Project

This is a simple C++ console application that manages a list of students and allows the user to search for a student by DNI and assign a grade.

---

## 🚀 Features

- 📚 Student struct management
- 🔍 Search by DNI
- ✏️ Grade assignment
- 📊 Data update in memory

---

## 🛠️ Technologies

- C++
- Standard Library (`iostream`, `string`)

---

## 💻 How it works

1. Program loads a list of students
2. User enters DNI
3. Program searches student
4. If found, user assigns a grade
5. Result is displayed

---

## 🎯 Learning Goals

- Practice structs in C++
- Implement linear search
- Modify data in arrays
- Work with basic algorithms
